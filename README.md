# Birthday Terminal Website

This is a special birthday website for Alex with a bash terminal theme.

## How to Use

Simply open `index.html` in your browser or visit the GitHub Pages URL.

### Available Commands

- `help` - Shows all available commands
- `msg <name>` - Displays a birthday message from someone
- `gallery` - Opens the photo gallery
- `clear` - Clears the terminal
- `about` - Information about this terminal
- `ls` - Lists available messages
- `whoami` - Shows current user
- `date` - Shows current date and time

## Adding Images to Gallery

The gallery **automatically detects** images in the `img/` folder!

Simply add your images with numbered filenames:

- `img/1.png` (always shown first)
- `img/2.png`
- `img/3.jpg`
- etc.

**Supported formats**: png, jpg, jpeg, gif, webp

The gallery will automatically find and display all numbered images in order. No need to edit any code!

## Customizing Messages

Edit `messages.json` to add or modify birthday messages.

## GitHub Pages Deployment

1. Push all files to the main branch
2. Go to repository Settings > Pages
3. Select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Save and wait for deployment

Your site will be available at: `https://31-01-loco.github.io`
