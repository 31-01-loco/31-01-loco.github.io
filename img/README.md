# Gallery Images

This folder contains the images for the birthday gallery.

## Adding Your Own Images

1. Add your images to this folder (supported formats: jpg, png, gif, webp)
2. Update the `galleryImages` array in `script.js` with your image filenames

Example:

```javascript
galleryImages = ["img/photo1.jpg", "img/photo2.jpg", "img/your-image.jpg"];
```
